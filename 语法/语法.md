


## str 与 bytes 转换

```py
# bytes object  
b = b"example"  

# str object  
s = "example"  

# str to bytes  
bytes([initializer[, encoding]])
bytes(s, encoding = "utf8")  
str.encode(s)  

# bytes to str  
str(b, encoding = "utf-8")  
bytes.decode(b)
```


## 向下取整

```py
a = 3.75
int(a)
# 3
```


## 四舍五入
```py
a=3.25
b=3.75
round(a)
round(b)
# 3.0
# 4.0
```
